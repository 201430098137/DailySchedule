# Git

## git 使用说明

目的：提高效率、注重规范、提升质量、优化记录、赏心悦目

注：属于进阶行为、不知道不影响日常使用、非必会。

## 背

### 空间

working directory -> staging area -> .git directory

### 文件

untracked -> unmodified -> modified -> staged

## 基

从 工作目录 添加 文件 A 进 暂存区

    git add A

把 暂存区 内容 提交 到 工作目录

    git commit -m "提交说明"
